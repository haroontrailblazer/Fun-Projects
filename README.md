## What Emilio Does

- Lets you enter **loan amount**, **interest rate**, and **duration**
- Calculates the EMI instantly
- Shows a **clean pie chart** to visualize:
  - Principal
  - Total Interest
  - Monthly Payment
- Smooth UI made with **Streamlit**
- Dark theme (because dark mode = good vibes)

Nothing too fancy ‚Äî just a clean, beginner-friendly tool that actually works
